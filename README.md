# Теория ULM/MAF → LAPL(A) или PPL(A)

Открытый русскоязычный курс строится в строгом порядке: сначала национальная
лицензия **ULM** с habilitación **MAF** для полётов только в Испании, затем
осознанный выбор дальнейшего маршрута **LAPL(A)** или **PPL(A)** по Part-FCL.
Общая теория изучается один раз, а различия последующих маршрутов выделяются
отдельно.

Нормативный снимок курса: **13 июля 2026 года**. Национальная испанская
лицензия ULM/MAF не даёт автоматического права выполнять международные полёты.

> Курс предназначен только для самостоятельного образования. Он не является
> официальной программой подготовки и не заменяет centro de formación de ULM,
> DTO/ATO, инструктора, AFM/POH конкретного воздушного судна, официальные
> материалы AESA или актуальную предполётную информацию.

## Проверить и собрать локально

Требуется Python 3. Создайте отдельное окружение и установите закреплённые
зависимости:

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
```

Единая команда проверки запускает тесты и строгую сборку MkDocs:

```bash
python scripts/validate_course.py
```

Для локального просмотра:

```bash
python -m mkdocs serve
```

Без установленного MkDocs валидатор всё равно запускает unit tests и явно
сообщает, что сборка сайта пропущена.
