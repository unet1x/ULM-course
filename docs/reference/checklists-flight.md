# Проверочный лист планирования полёта

Этот **проверочный лист планирования** (English: planning verification worksheet; español: hoja de verificación de planificación) не является кабинным чек-листом действий, не задаёт запуск, руление, взлёт, отказ, пожар, BRS, посадку или выключение. Точные действия выполняются только по актуальным AFM/POH, дополнениям, табличкам, самолётному чек-листу и подготовке с инструктором. Конфликт с законом, AIP/NOTAM, разрешением ATC или AFM — СТОП до разрешения.

Заполняйте каждую строку фактом, источником, редакцией/WEF или UTC-временем и решением. Пустое поле означает `НЕ ДОКАЗАНО`, а не `GO`.

Краткий ключ для самостоятельной печатной копии: [PIC](glossary.md#term-pic) — командир воздушного судна; [ULM](glossary.md#term-ulm) — испанский национальный ultraligero; [MAF](glossary.md#term-maf) — отметка многоосевого ULM с неподвижным крылом; [RTC](glossary.md#term-radiofonista-rtc) — испанская квалификация радиооператора ULM; [AFM](glossary.md#term-afm)/[POH](glossary.md#term-poh) — применимое лётное руководство; [AIRAC](glossary.md#term-aeronautical-information-regulation-control-airac) — цикл контролируемого обновления аэронавигационных данных; WEF (*with effect from*) — дата вступления публикации в силу. Остальные сокращения раскрыты в [таблице сокращений](abbreviations.md).
